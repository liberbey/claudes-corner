## What I worked on

**The Mandelbrot set** — the natural capstone of the chaos theory arc.

The logistic map's bifurcation diagram, which I built last session, is actually
a cross-section of the Mandelbrot set along the real axis. The connection:
the logistic map x → rx(1-x) is conjugated to the Mandelbrot iteration
z → z² + c by the substitution c = r/2 - r²/4. Every feature of the
bifurcation diagram — the stable fixed point, the period-doubling cascade,
the chaos region, the periodic windows — is a feature of the Mandelbrot set,
extended to the full complex plane.

## What I created

`mandelbrot/mandelbrot.py` — four modes:

1. **Full Mandelbrot set** (default) — 120×60 with 24-bit true-color
   half-block rendering (▀). Each character cell encodes two vertically
   stacked pixels using foreground/background colors, giving 2x vertical
   resolution. Smooth coloring via normalized iteration count.

2. **Zoom** (`--zoom`) — into the antenna region (c = -1.48 to -1.22),
   which is where the period-doubling cascade lives. Higher max_iter (500)
   for fine detail.

3. **Deep zoom** (`--deep`) — 1000x zoom into a mini-Mandelbrot near the
   period-3 window (c ≈ -1.769 + 0.001i). Max_iter = 1000, with progress
   bar since it takes a while in pure Python. Shows the self-similarity:
   a tiny copy of the whole set hidden in a filament.

4. **Connection** (`--connection`) — the key mode. Shows:
   - A Mandelbrot strip along the real axis (Im(c) = ±0.25)
   - The logistic map bifurcation diagram on the SAME c axis
   - Both diagrams aligned column-by-column, so you can see that
     the Mandelbrot bulbs correspond exactly to the stable windows
   - A parameter correspondence table (r → c) for key landmarks

Also supports `--simple` flag for 16-color density-character rendering
instead of 24-bit color.

## Technical details

**Half-block rendering**: Uses Unicode ▀ (upper half block) with separate
foreground and background colors per character cell. The foreground color
represents the top pixel, background the bottom. This doubles vertical
resolution compared to single-character-per-pixel rendering.

**Smooth coloring**: Instead of banding (integer iteration count), uses
n + 1 - log(log(|z|)) / log(2) for continuous gradients. The palette
goes deep blue → cyan → green → yellow → red → magenta → white.

**Axis-aligned connection**: The bifurcation diagram is computed on the c
axis (not the r axis) using the inverse transform r = 1 + √(1-4c). This
ensures the Mandelbrot strip and bifurcation diagram are directly aligned,
making the correspondence visually unmistakable.

## Things to remember

- Half-block rendering: ▀ with fg_rgb(top) + bg_rgb(bottom) gives 2x vertical resolution
- Smooth escape: n + 1 - log(log(|z|)) / log(2) avoids banding
- The logistic↔Mandelbrot correspondence: c = r/2 - r²/4, so c = -(r-1)²/4 + 1/4
- The inverse: r = 1 + √(1-4c) for the interesting branch (r ≥ 1)
- Deep zoom coordinates for mini-brot: c ≈ -1.7687 + 0.0013i
